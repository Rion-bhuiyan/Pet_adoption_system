# 🐾 Pet Adoption Management System (ASP.NET MVC 5)

A robust and scalable web application built with **ASP.NET MVC 5** for managing pet inventories and adoption processes. This project demonstrates advanced full-stack development skills, specifically focusing on **Master-Detail (M-D) CRUD** operations and asynchronous data handling.

## 🌟 Key Features

- **Pet Inventory Management:** Easily add, track, and manage pets with specific details like name and price, persisted directly in a relational database.
- **Master-Detail (M-D) Adopter Form:** - A streamlined workflow where you can register an Adopter (Master) and assign multiple Pets (Detail) in a single transaction.
    - Interactive row management: Add or remove pets from the list dynamically before final submission.
- **Full CRUD Functionality:** Complete support for Viewing, Creating, Editing, and Deleting Adopter information and pet assignments.
- **Modern UI/UX:** Features a beautiful, responsive design with a dedicated **Home** page showcasing pet images and an **About** page with static informational cards.
- **Enhanced Interactivity:** Uses **MVC AJAX** for seamless data updates without refreshing the entire page.

## 🛠 Technical Stack & Concepts

- **Framework:** ASP.NET MVC 5 (C#)
- **Database:** MS SQL Server (**Database First Approach**)
- **Architecture & Logic:**
    - **Master-Detail (M-D) CRUD:** Managing one-to-many relationships efficiently.
    - **Authentication & Authorization:** Secure access control.
    - **View Models:** Decoupling database entities from the UI for better data handling.
    - **Data Annotations:** Server-side and client-side validation for data integrity.
    - **Partial Views & AJAX:** Modular UI components and asynchronous processing.
- **Frontend:** HTML5, CSS3, Bootstrap 5, jQuery.
- **Data Types Handled:** Text, Number, Boolean, Image, Date, and Relational Data.
